  • vim
    Es un editor de texto alternativo minimalista y se abre:
    ~$vim Nombre del Archivo

  • nano
    El editor de texto se abre con:
    ~$nano Nombre del Archivo
    Para guardar el documento es con: Ctrl + o

    Para salir del programa es con: Ctrl + x

    Para realizar una Busqueda es con: Ctrl + q

    Para cancelar la busqueda es con: Ctrl + c

  • micro
    El editor de texto se abre con:
    ~$micro Nombre del Archivo
    Para guardar el documento: Ctrl + s

    Para regresar una modificacion hecha en el momento: Ctrl + z

    Para cortar una linea es con: Ctrl + c

    para quitar o mostrar el enumerado de las lineas: Ctrl + r

    Para salir del editor es: Ctrl + q

  • cat
    cat es un comando para previsualizacion de un documento
    el cual se ejecuta con:
    ~$cat Nombre del Archivo
